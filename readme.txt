Put an affiliate leaderboard for the CrossFit Games Open 2018 on your website.

== Description ==
**WP CFG Leaderboard** provides a shortcode to put your affiliate leaderboard for the CrossFit Games Open 2018 into your WordPress site.

DISCLAIMER: The plugin and its author are in no way associated with or endorsed by CrossFit Corp or the CrossFit Games. The plugin relies on the website API used by the official Games Website which could be discontinued or changed at any time.

**Features**

* **Shortcode to display leaderboard**: shows the affiliate leaderboard in posts or pages.
* **Lightweight leaderboard page template**: Can be used to create a blog page that displays only the leaderbaord using the shortcode. 

== Installation ==
**Installation**:

* Install and Activate the Plugin via the WordPress Plugin menu
* Under Settings / CFG Leaderboard enter the affilate id for your affiliate. Find your ID by going to the Games website, search your affilate profile and use the number at the end of the webpage address.
* More instructions on using the shortcode can be found in the documentation section under settings.
